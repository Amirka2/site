Для того, чтобы запустить сайт локально, нужно: 
скачать все файлы с гитхаба,
запустить файл index.html с помощью браузера.
Личный кабинет доступен пока только через открытие напрямую из файлов(LK.html)
На другие страницы можно попасть с главной, так же css и js связаны со страницами, а картинки используются.
